Cloud query interface

You can send a query to the cloud by using a MongoDB like query:

Query help

You can use the following fields in your query

  • uploadId int
  • uploadDate int
  • uploadIp string
  • storagePath string
  • storageFile string
  • storageHttp string
  • storageAddress string
  • jobsCompleted int
  • jobsErrors int
  • jobsSuccess int
  • jobsTotal int
  • fileSize int
  • storageSize int
  • mimeType string
  • taskId int
  • taskStatus string
  • taskType string
  • taskPreset string
  • taskSize int
  • taskExtension string
  • taskPriority int
  • taskStartDate int
  • taskEndedDate int
